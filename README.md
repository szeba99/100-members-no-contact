### INFORMATION FOR DEVELOPERS

This project is using Godot 4.7-stable which can be downloaded for multiple platforms from the [official Godot engine website]([https://godotengine.org/](https://godotengine.org/download/archive/4.7-stable/))

Please make sure you have a basic understanding of git/github and forks/pull requests before contributing. It will save us a lot of headaches and time handling merge conflicts and such.

Additional project information can be found in the "Some Info" folder.
