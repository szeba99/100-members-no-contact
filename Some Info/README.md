# 100-members-no-contact

A game for the "Very serious game jam" hosted by Juniper Dev.

We're making a Beyblade roguelike/metroidvania deckbuilder RPG (Or Beyblade builder Idk the terminology). Basically a Beyblade game with a story. No more explanations, Go crazy!!!
